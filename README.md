# Python-Portfolio-Optimiser-
This project explores the fundamentals of portfolio diversification and risk-return optimisation using historical financial data and simple Python for educational and illustrative purposes.
The project imports historical price data, calculates daily returns, estimates expected returns for each asset, computes the covarianve matrix between assets and then simulates different portfolio allocations based on user's chosen risk tolerance. 
This project uses Pandas, Numpy and Matplotlib.
